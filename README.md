# sprint_7_project
Final project - Sprint 7

Esta aplicación web, desarrollada con Streamlit, permite explorar de manera interactiva un conjunto de datos de anuncios de venta de vehículos en Estados Unidos. El panel ofrece visualizaciones sencillas creadas con Plotly, como histogramas y gráficos de dispersión, que ayudan a identificar patrones y distribuciones dentro del dataset.

La aplicación carga los datos desde un archivo CSV y permite al usuario generar gráficos con un clic, facilitando una experiencia intuitiva para inspeccionar las variables principales del dataset.
